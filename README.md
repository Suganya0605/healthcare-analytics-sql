# Healthcare Analytics with SQL

## Project Overview

This project focuses on analyzing healthcare data using SQL to extract meaningful insights about patients, doctors, appointments, diagnoses and medications.

The project demonstrates advanced SQL concepts including:

* Inner Joins
* Left Joins
* Aggregate Functions
* Window Functions
* Conditional Logic
* Subqueries
* Date Functions
* Complex Data Analysis

## Database Tables

1. Patients
2. Doctors
3. Appointments
4. Diagnoses
5. Medications

## Key Business Questions Answered

* Completed appointment analysis
* Doctor workload analysis
* Patient visit frequency ranking
* Age group categorization
* Medication duration analysis
* Unique patient count per doctor
* Contact number filtering
* Insulin prescription analysis
